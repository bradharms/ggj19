# GGJ 2019 Project

## Synopsis

Tower defense in first-person 3D, possibly VR if enough time

## Running / Installing

- Install [Node.js](nodejs.org)
- Clone the project
- Open command prompt
- CD to project directorty
- Run `npm install`
- Run `npm start`
- Open web browser to http://localhost:8080
